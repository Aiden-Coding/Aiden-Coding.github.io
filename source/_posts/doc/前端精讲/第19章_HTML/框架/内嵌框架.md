
# 内嵌框架

内嵌框架是用于在网页中嵌入一个网页并让它在网页中显示. 

添加内嵌框架的语法: 


|<iframe src=" URL "></iframe> |

URL 指定独立网页的路径. 




案例： 

展示图书： 




书籍展示首页： 




1.  <!DOCTYPE html>
2.  <html>
3.          <head>
4.                  <meta charset="UTF-8">
5.                  <title></title>
6.          </head>
7.          <body>
8.                  <iframe src="书籍导航页面.html" height="700px"
    width="30%"></iframe>
9.                  <!--内嵌框架-->
10.                 <iframe name="iframe_my" width="67%" height="700px"
    src="img/think in java.jpg"></iframe>
11.         </body>
12. </html>
