
# Java中的运算符

【1】Java 语言支持如下运算符： 

- 算术运算符    

+，-，*，/，%，++（自增），--（自减） 

- 赋值运算符  

=   

- 扩展赋值运算符 

+=，-=，*=，/= 

- 关系运算符 

  >，<，>=，<=，==，!= 

- 逻辑运算符 

&，|， &&，||，!，^ 

- 位运算符 

 &，|，^，~ ， >>，<<，>>> (了解！！！) 

- 条件运算符 

  ？：
