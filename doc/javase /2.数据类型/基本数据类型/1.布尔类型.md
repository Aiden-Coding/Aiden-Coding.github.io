
# 布尔类型

boolean类型有两个常量值，true和false，在内存中占一位（不是一个字节），不可以使用 0 或非 0 的整数替代 true 和 false
，这点和C语言不同。 boolean 类型用来判断逻辑条件，一般用于程序流程控制 。
