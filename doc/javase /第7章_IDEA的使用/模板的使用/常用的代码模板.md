
# 常用的代码模板

【1】模板1： main方法： 

main  或者 psvm 




【2】模板2：输出语句： 

sout   或者   .sout 

一些变型：
soutp:打印方法的形参 

soutm:打印方法的名字 

soutv:打印变量 




【3】模板3： 循环 

普通for循环：   fori（正向）   或者   .fori （正向）   . forr(逆向) 

增强for循环：  iter  或者  .for 

（可以用于数组的遍历，集合的遍历） 




【4】模板4： 条件判断 

ifn 或者  .null ：判断是否为null  （if null） 

inn 或者 .nn ：判断不等于null   (if not null) 




【5】模板5： 属性修饰符： 

prsf : private static final 

psf  :public static final
