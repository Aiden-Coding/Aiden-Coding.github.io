
# 什么是HTML

【1】HTML 指的是超文本标记语言: HyperText Markup Language 

超文本： 

普通人 vs  超人（比普通的人厉害） 

     普通文本  vs 超文本(比普通的文本厉害) 

标记： 

    标记=标签 

    标签：<html> <body> <head> 由尖括号包围起来的关键词 

分类：双标记标签/封闭类型标签  <p></p>     单标记标签/非封闭类型标签   <br/> 

语言：HTML是一个描述网页的语言 




总结：学习HTML就是学习各种各样的标签，然后组成一个页面，这个页面可以被浏览器解析，解析完以后可以在浏览器中将页面进行展示。
