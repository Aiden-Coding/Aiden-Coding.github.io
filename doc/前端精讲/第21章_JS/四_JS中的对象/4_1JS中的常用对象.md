**Date对象**


Date 对象用于处理日期与时间。 

创建 Date 对象： new Date() 

以下四种方法同样可以创建 Date 对象： 

var d = new Date();

var d = new Date(milliseconds);

var d = new Date(dateString);

var d = new Date(year, month, day, hours, minutes, seconds, milliseconds);
